# Password Security & Cracking Analysis

## 📌 Description
This project evaluates password strength and demonstrates how weak passwords can be cracked using simple techniques.

## 🚀 Features
- Password Strength Checker
- Strong Password Generator
- Dictionary Attack Simulation

## 🛠️ Technologies
- Python
- Regex

## ▶️ How to Run
1. Open project in VS Code
2. Run:
   python main.py

## 📊 Output
- Shows password strength (Weak / Medium / Strong)
- Gives suggestions to improve password
- Simulates password cracking attempts

## 🔐 Learning Outcome
- Understand password vulnerabilities
- Learn importance of strong authentication
