# Keylogger Detection & System Monitoring

## 📌 Description
This project detects suspicious processes such as keyloggers by monitoring system activity.

## 🚀 Features
- Process Monitoring
- Suspicious Process Detection
- Alert Logging

## 🛠️ Technologies
- Python
- psutil library

## ▶️ How to Run
1. Install dependencies:
   pip install psutil

2. Run:
   python main.py

## 📊 Output
- Shows suspicious processes
- Logs alerts in alerts.log

## 🔐 Learning Outcome
- Understand malware detection basics
- Learn system monitoring using Python
